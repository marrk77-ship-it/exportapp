# 管理画面 利用ガイド

## 🔑 アクセス方法

### 方法1: 通常画面からアクセス（推奨）

1. **通常画面にログイン**
   - https://exportapp-tw.tech にアクセス
   - 管理者アカウントでログイン:
     - ログインID: `admin` / パスワード: `admin2024`
     - または `client1` / `password123`

2. **管理画面ボタンをクリック**
   - ログイン後、画面右上に「管理画面」ボタンが表示されます（管理者のみ）
   - このボタンをクリックすると管理画面に移動します

### 方法2: 直接アクセス

1. **管理画面URLに直接アクセス**
   - https://exportapp-tw.tech/admin にアクセス

2. **ログイン画面が表示される**
   - ログインID: `admin`
   - パスワード: `admin2024`
   - または `client1` / `password123`

3. **管理ダッシュボードが表示される**

## ⚠️ 重要な注意事項

### クッキーとセッションについて

管理画面は**セッションクッキー**を使用してユーザーを認証します。

- **通常画面でログイン済みの場合**: そのまま管理画面にアクセスできます
- **未ログインの場合**: 管理画面のログイン画面が表示されます

### ブラウザのクッキー設定

以下の設定が必要です：
- ✅ **クッキーを有効にする**: ブラウザの設定でクッキーを許可
- ✅ **サードパーティクッキーをブロックしない**: exportapp-tw.tech のクッキーを許可
- ✅ **プライベートブラウジングモードを使用しない**: 通常モードで利用

### トラブルシューティング

#### ログインできない場合

1. **ブラウザのキャッシュとクッキーをクリア**
   - Chrome: 設定 → プライバシーとセキュリティ → 閲覧履歴データの削除
   - Firefox: オプション → プライバシーとセキュリティ → データを消去

2. **別のブラウザで試す**
   - Chrome、Firefox、Edge、Safari などで試してみてください

3. **シークレットモード/プライベートブラウジングを避ける**
   - 通常モードで開いてください

4. **パスワードを確認**
   - ログインID: `admin`
   - パスワード: `admin2024` （正確に入力）

#### 「セッションチェックエラー」が表示される場合

これは、まだログインしていないか、セッションが期限切れの場合に表示されます。

**解決方法:**
1. ログイン画面で改めてログイン
2. または、通常画面（https://exportapp-tw.tech）でログインしてから管理画面に移動

## 📊 管理画面の機能

### 1. システム統計ダッシュボード
- 総ユーザー数
- 総CSV件数
- データベースサイズ

### 2. ユーザー管理
- **ユーザー一覧表示**: ID、ログインID、名前、ロール、CSV件数、作成日
- **新規ユーザー作成**: 「新規ユーザー」ボタンから作成
- **パスワードリセット**: 🔑アイコンをクリック
- **ユーザー削除**: ❌アイコンをクリック（自分自身は削除不可）

### 3. データ管理
- **CSVデータ表示**: 👁アイコンでプレビュー
- **CSVデータ削除**: 🗑アイコンで削除

### 4. 操作ログ
- 管理者の操作履歴を最新100件表示
- 操作者、操作内容、対象ユーザー、IPアドレス、日時

## 🔒 セキュリティ

### 管理者権限

- `role='admin'` のユーザーのみアクセス可能
- 一般ユーザー（`role='user'`）は管理画面にアクセスできません

### 操作ログ

すべての管理操作は記録されます：
- ユーザー作成
- ユーザー削除
- パスワードリセット
- CSVデータ削除

### セッション管理

- セッション有効期限: 7日間
- HTTPOnly: ✅（JavaScriptからアクセス不可）
- Secure: ✅（HTTPS接続のみ）
- SameSite: Strict（CSRF攻撃を防止）

## 📞 サポート

問題が解決しない場合は、以下の情報を共有してください：
- ブラウザの種類とバージョン
- ブラウザのコンソールに表示されるエラーメッセージ（F12キーを押してConsoleタブを確認）
- スクリーンショット
